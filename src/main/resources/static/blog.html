徐巧芯提發現金3大理由：我家以前開早餐店「振興券無助小店家」

近日行政院提出「振興五倍劵」的規劃，預計9月推出，重施去年三倍券的模式，五倍券改為讓民眾用1000元換5000元； 台北市議員徐巧芯今（13）日表示，她家以前開早餐店， 還算了解小店家的營運模式，她提出發現金比發消費券、五倍券都好的3大理由， 並質疑發券能幫助到真的需要幫忙的小店家嗎？是要發大賣場振興券嗎？ 她在臉書發文表示，為什麼發「現金」比發「消費券」好，又比要先拿1000出來「參與」的五倍券好？ 她說，家裡以前是開早餐店的，她還算蠻了解小店家的營運模式。疫情期間不能內用，只能改外帶， 小店家苦不堪言，此波他們應該也是主要的振興目標。因此提出3個要點關於「為什麼發現金，比發消費券或振興券好」。 
p { color: red}
第一點她表示，應該考量小店家的現金週轉問題。 因為經過這幾個月的煎熬，小店家的現金週轉早已快撐不下去了。 每天的食材成本、房租、水電都是要用現金支付，而疫情期間生意受到影響，許多店家快拿不出來了。 她表示，現在要振興，讓消費者積極消費，但等到的券要拿去兌換成現金，難以收到後就使用，例如繳交水電費等，這樣仍然增加他們週轉負擔。
